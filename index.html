Typography

Headings should in Lato font 400 weight, no additional styling. H2 should be main color. paragraphs are in Merriweather font 400 weight. Paragraphs have option of italicized in 400 weight only. For desktops heading size should be 3rem for h1, 2.5rem for h2, 2rem for h3. Sizes should scale for smaller screen resolutions.
